# UI/UX SPEC — BOOKINDER

## 🎨 Design Philosophy

* Minimal
* Editorial (like shortform.com)
* Spacious
* Calm and premium
* No clutter

---

## 🌈 Color System

* Background: #F8F5F0 (cream)
* Primary: #6B4F3A (soft brown)
* Dark: #2B2B2B
* Accent: #A67C52
* Border: #E5DED6

---

## 🔤 Typography

* Headings: Serif (Playfair Display or similar)
* Body: Sans-serif (Inter)

---

## 🏠 LANDING PAGE

### Layout

* Full viewport height
* Centered vertically and horizontally
* Max width: 700px

---

### Elements

#### Heading

* Large serif font
* Size: ~48px
* Text:
  “Find Any Book PDF Instantly”

#### Subtext

* Light gray
* Size: 16px

---

### Search Bar

* Height: 60px
* Rounded: 999px
* Padding: px-6
* Background: white
* Shadow: soft

#### Interaction:

* Focus → glow border (brown)
* Hover → slight lift

---

### Button

* Rounded
* Brown background
* White text
* Hover:

  * Slight scale
  * Darker brown

---

## 🔄 ANIMATIONS

* Page load → fade + slide up
* Input focus → smooth glow
* Results → staggered fade-in
* Button hover → scale 1.03
* Cards → hover lift + shadow

---

## 📄 SEARCH PAGE

### Layout

* Top search bar (sticky)
* Results below
* Max width: 900px
* Centered

---

## 📦 RESULT CARD

### Structure

* White background
* Rounded-xl
* Padding: p-5
* Border subtle

---

### Content

1. Title

   * Bold
   * Dark color

2. Domain

   * Small text
   * Gray

3. Snippet

   * 2 lines max

4. Actions

   * Open PDF (primary button)
   * Copy Link (secondary)

---

### Hover

* Shadow increases
* Slight upward movement

---

## ⏳ LOADING

* Skeleton cards
* Animated shimmer

---

## ❌ EMPTY STATE

* Message:
  “No PDFs found. Try a different search.”

---

## 📱 RESPONSIVE

* Mobile:

  * Full width cards
  * Smaller text
  * Stacked layout

---

## ⚠️ STRICT UI RULES

* No extra colors
* No gradients
* No clutter
* Maintain spacing consistency
* Every element must breathe

---

This UI must feel like a premium reading product.
