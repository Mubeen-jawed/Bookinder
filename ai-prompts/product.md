# PRODUCT SPEC — BOOKINDER

## Overview

Bookinder is a minimal SaaS web app that helps users find publicly available PDF versions of books.

## Core Functionality

* User enters book name
* App searches:
  "[book name] filetype:pdf"
* Uses Google Custom Search API
* Displays results cleanly

---

## Features

* Search input
* Results listing
* PDF open button
* Source display
* Snippet preview
* Loading states
* Error states
* Responsive design

---

## Pages

1. Landing Page (`/`)
2. Search Results (`/search?q=`)

---

## Target UX

* Fast
* Clean
* Elegant
* Distraction-free
